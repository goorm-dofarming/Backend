- OAuth를 사용한 소셜 회원가입/로그인 구현
- Kafka 클러스터 구현
- 채팅 서버, 알림 서버 연동 및 구현
- QueryDsl을 사용하여 리뷰 및 좋아요 리스트 정렬 기능 구현
- 전체 엔티티 작성 및 연관관계 설정
- 공통 기능 및 글로벌 에러 처리 작성
- ERD 및 시스템 아키텍처 작성
